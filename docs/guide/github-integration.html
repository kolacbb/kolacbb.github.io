# Integrate Github hosting services to AppLocalize workflow

Integrations with Git hosting services allow you to easily transfer localization files between the localization projects hosted in your AppLocalize account or organization and your repositories.

To get started, you'll need to access the Integrations section in the navigation bar and connect AppLocalize to your code hosting platform of choice.
